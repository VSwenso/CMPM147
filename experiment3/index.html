Experiment 3 - Alternate Worlds

Description

The goal of this assignment is to create a randomly generated overworld map and dungeon map

Technical

For this project I used the help of chatgpt, friends advice and got some inspiration from Alex's code for how to get both biomes on the same porfolio page.

Reflection

*using tiles from Micro Tileset - Overworld & Dungeon by thkaspar* For the project overall i spent a ton of time working on it and i will be honest when i started i had no idea where to begin. While i did have some experience working with Tiled, this was my first time working with a tileset in p5.js. At first I was having a ton of issues with getting both of my biomes on my page, but one of my group members posted how they were able to get theirs working so used that as a guide for my work.
